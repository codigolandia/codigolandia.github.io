Live Quest

O projeto Live Quest é o nosso atual projeto em desenvolvimento. O objetivo deste projeto é facilitar uma forma interativa de aprendizagem de programação.

Os participantes da transmissão ao vivo irão desbloquear recursos nessa quest ao completarem desafios de lógica de programação, em diferentes linguagens. Durante as transmissões, iremos ajudar no desenvolvimento destes desafios para quem precisar daquela força!
